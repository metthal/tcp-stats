Description

TCP Stats is application capable of analyzing TCP stream stored in pcapng format and storing the analyses results in JSON format. The application is shipped with this web page, which visualize output in JSON file into HTML form.

Dependencies

  • g++/clang++ with C++11 support
  • CMake 2.8+
  • libpcap 0.8+
  • jsoncpp 0.6+

Compilation

TCP Stats can be compiled either by running script run-build.sh shipped with the application or manually by configuring CMake.

  • Compiling with script - run ./run-build.sh in projects home directory.
  • Manual configuration - run cmake in project home directory or any other CMake tool

Running

TCP Stats has only one startup argument and that is input pcap file with valid TCP stream. TCP stream is considered valid if

  • Only single stream is present in the input file.
  • Stream is captured with its handshake, not necessarily with the termination of the connection.

Visualization

Visualization of output data on this web page is performed using several libraries.

  • jQuery - prerequisite of other libraries
  • Bootstrap - layout, design and UI elements
  • Highcharts - library for plotting charts
